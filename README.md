# 문서 변환기 (HWPX ↔ DOCX)

아래아한글(HWPX/HWP)과 MS 워드(DOCX)를 **상호 변환**하는 크롬 확장 + 로컬 도우미.

## 구성
- **`extension/`** — 크롬 웹스토어용 확장 (브라우저 단독 텍스트 변환 + 로컬 도우미 연동)
- **`helper/`** — 로컬 도우미 (설치된 한/글을 COM으로 구동해 **원본급 변환**). 사용자 PC에서만 동작, 파일 외부 유출 없음
- **`참고파일/`** — 개발·참고용 (구 백엔드 소스 등, 배포 아님)

## 동작 방식
1. 확장 설치(웹스토어) → 기본(텍스트) 변환 즉시 사용
2. 원본급 품질을 원하면 **로컬 도우미 1회 설치** → 확장이 `127.0.0.1`의 도우미를 감지해 한/글 엔진으로 변환
3. 변환은 전부 사용자 PC 안에서 처리 (서버 불필요, 파일 유출 없음)

## 상태
- 확장(브라우저 변환): 동작 ✅
- 한/글 COM 원본급 변환: 검증 완료 ✅ (HWP↔DOCX↔HWPX, 이미지·표 보존)
- 남은 과제: 도우미 무인 구동 시 한/글 자동화 보안모듈(FilePathCheckerModule) 등록
